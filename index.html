Welcome to Visual Servoing for Robot Manipulator

Visual servoing adalah sebuah teknik kontrol yang menggunakan umpan balik visual (biasanya dari kamera) untuk mengontrol pergerakan sebuah robot manipulator. Teknik ini memanfaatkan informasi visual untuk menyesuaikan posisi dan orientasi robot agar mencapai tujuan tertentu. Visual servoing sangat populer dalam aplikasi robotika karena memberikan kemampuan adaptasi terhadap lingkungan yang dinamis atau tidak terstruktur.

mempelajari/memahami design robot, program robot, app pergerakan, dan desain wiring

Our resources include the following:

  • Design: Desain robot arm 6 DOF (Degrees of Freedom) dengan visual servoing untuk manipulasi robot adalah sistem yang dirancang untuk memungkinkan robot memanipulasi objek atau melakukan tugas dengan presisi tinggi menggunakan umpan balik visual.
  • Program: Karena kodenya sedikit lebih panjang, saya akan memposting kode sumber program dalam beberapa bagian dengan deskripsi untuk setiap komponen agar lebih mudah dipahami. Untuk kode lengkap bisa download link di bawah berikut.
  • App penggerakan robot: Aplikasi ini dirancang untuk mengontrol lengan robot (robot manipulator) dengan 6 Degrees of Freedom (DOF). Sistem ini berfungsi untuk:
    • Kontrol Manual Robot Arm: Memungkinkan pengguna mengatur posisi setiap bagian lengan robot secara manual menggunakan slider yang ada.
    • Perekaman dan Pemutaran Gerakan: Memberikan kemampuan untuk merekam gerakan yang dilakukan pada robot, lalu memutarnya kembali untuk tugas-tugas otomatisasi.
    • Integrasi Visual Servoing: Meskipun tidak terlihat eksplisit dalam antarmuka, aplikasi ini mendukung visual servoing untuk meningkatkan presisi gerakan.
  • Wiring: Rangkaian untuk mengontrol robot manipulator berbasis ESP32 Dev Kit dengan menggunakan servo motor. Komponen-komponennya bekerja sebagai berikut:
    • ESP32 mengontrol pergerakan servo berdasarkan sinyal PWM (Pulse Width Modulation).
    • Step-down converter memastikan semua komponen mendapat tegangan yang sesuai (5V).
    • Kombinasi ini memungkinkan sistem robot manipulator bekerja dengan efisien dan aman.
